# unit_tests_example PROJECT

Tested on Ubuntu 16.04, GCC 5.4.0, CMake 3.5.1

[![Build Status](https://travis-ci.org/gocarlos/unit_tests_example.svg?branch=master)](https://travis-ci.org/gocarlos/unit_tests_example)


## USAGE - RUN
In the project root:
* rm -rf build/*
* mkdir build
* cd build
* cmake ..
* make
* ./main      	% executable
* ./unit_test	% unit test
